# OSRS Quest Item Checker

A web application to help Old School RuneScape players determine which quest items they should keep in their bank.

## Features

- **Fuzzy Search** - Find items even with typos or partial names
- **Quest Item Lookup** - Instantly check if an item is needed for future quests
- **Autocomplete** - Quick item suggestions as you type
- **Comprehensive Database** - 16,549 items with 129 quest-essential items tracked

## Tech Stack

- [Next.js 15](https://nextjs.org/) - React framework with App Router
- [TypeScript](https://www.typescriptlang.org/) - Type-safe JavaScript
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [Fuse.js](https://www.fusejs.io/) - Lightweight fuzzy-search library

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
